#SmartLib

##07/01/2016
	功能：该App实现一个简单的测试，即根据输入字段，从图书馆获取搜索Response，并显示在TextView中
	主要需要改进的核心功能：
	1、搜索结果需要用正则表达式进行过滤，并建议过滤后进行对象化，以便调用;
	2、目前urlString对大部分参数都固定化，可以考虑根据前端需求，接收部分参数数组后进行自定义url生成


##07/02/2016

###仍需改进部分：
